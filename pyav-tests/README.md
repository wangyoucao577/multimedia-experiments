# pyav

## Installation 

```bash
conda create -n pyav python=3.11
conda activate pyav

# install pyav
conda install av -c conda-forge

# install matplot 
python -m pip install -U pip
python -m pip install -U matplotlib
```

## Usage

```bash
python probe_durations.py -i test.mp4
python plot-av.py -i test.mp4
```
